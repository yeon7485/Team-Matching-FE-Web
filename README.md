## 💻 구현 사항

- 회원가입, 로그인/로그아웃
- 팀 매칭
  - 팀 생성/수정/삭제 기능, 팀 검색 기능, 가입 신청 기능, 가입 신청 조회 및 수락/거절 기능
- 자유게시판
  - 게시판 내 검색 기능, 글 생성/수정/삭제 기능, 댓글 작성/삭제 기능
- 마이페이지
  - 작성한 글/댓글 조회 기능, 참여 중인 팀 조회, 가입 신청 중인 팀 조회 및 신청 취소 기능
- 팀 전용 페이지
  - 팀 정보 조회/수정, 팀 탈퇴, 삭제 기능
